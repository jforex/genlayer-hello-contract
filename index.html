Onchain AI Fact-Checker

Submit any claim. Five independent AI validators fetch live web data and reach consensus on the verdict — stored permanently on-chain. No oracle needed.

Try it in GenLayer Studio View on GitHub
How it works
1
Submit a claim
Call verify_claim() with any claim and an optional evidence URL on the GenLayer blockchain.
2
AI fetches + analyzes
5 independent AI validators fetch live web data on-chain and independently evaluate the claim.
3
Consensus verdict
Validators reach consensus via Optimistic Democracy. The verdict is stored permanently on-chain.
Example verdicts from the contract
"The Earth is flat"
No evidence URL · 5 validators · Consensus reached
False Overwhelming scientific evidence from geology, physics, and space exploration confirms the Earth is an oblate spheroid, not flat.
"Bitcoin is the largest cryptocurrency by market cap"
Evidence: coinmarketcap.com · Live web data fetched on-chain · 5 validators
True According to live data fetched from CoinMarketCap, Bitcoin holds the largest market capitalization among all cryptocurrencies.
"Satoshi Nakamoto's true identity is known"
No evidence URL · 5 validators · Consensus reached
Uncertain Despite various claims and investigations, the true identity of Bitcoin's creator remains officially unverified and disputed.
